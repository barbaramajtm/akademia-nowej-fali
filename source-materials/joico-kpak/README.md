## Rdzeń merytoryczny (O-W-R-N)

1. **O**czyszczanie — osady, chlor, minerały, resztki stylizacji  
2. **W**ygładzanie — pH / powierzchnia łuski  
3. **R**ekonstrukcja — składniki proteinowo-keratynowe / aminokwasy  
4. **N**awilżenie — miękkość, elastyczność, połysk  

Komunikacja: uczciwe obietnice (miękkość, gładkość, sprężystość) — bez „leczenia” włosa i bez trwałej odbudowy 100%.
