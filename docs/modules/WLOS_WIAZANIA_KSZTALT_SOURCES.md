## 1. Zakres audytu

### 1.1 Cel

Ustalić, czego **wolno uczyć** w module o wiązaniach i zmianie kształtu wyłącznie na podstawie lokalnych źródeł projektu — bez domysłów i bez powielania lekcji już zrealizowanych.

### 1.2 Wiedza już nauczona (nie powtarzać jako nowej)

| Lekcja / moduł | Co uczennica już wie | Konsekwencja dla tego modułu |
|----------------|----------------------|------------------------------|
| `wlos-z-czego-sklada-sie` | Skład: keratyna, woda, lipidy; ostrożne % | Nie powtarzać listy składu |
| `wlos-oslonka-kora-rdzen` | Osłonka / kora / rdzeń | Nie uczyć warstw od zera; najwyżej krótkie interleaving |
| `wlos-keratyna-rusztowanie` | Keratyna = białko / rusztowanie; pierwiastki C O N H S; teaser mostków siarczkowych | Nie powtarzać definicji keratyny; **rozwijać** rolę wiązań |
| Moduł `ph` (m.in. `ph-jak-wplywa-na-wlos`, `ph-produkty-zakwaszajace`) | pH a osłonka; utrwalacz odbudowuje wiązania siarkowe (wzmianka) | Nie uczyć skali pH; przy trwałej **pogłębić** proces, nie „odkrywać od zera” utrwalacza |
| N1–N20 | Numeracja farb, diagnoza koloru | Poza zakresem tego modułu |

### 1.3 Kierunek dydaktyczny (mapa logiczna)

budowa/keratyna (już znane)  
→ co utrzymuje kształt  
→ rodzaje wiązań  
→ co wpływa na poszczególne wiązania  
→ woda / wilgoć / temperatura → zmiana czasowa  
→ dlaczego fryzura po modelowaniu się zmienia  
→ różnica czasowa vs chemiczna  
→ wiązania istotne dla trwałej zmiany  
→ redukcja / zmiana / utrwalenie (gdy źródła pozwalają)  
→ zastosowanie w pracy fryzjera

## 4. Luki i sprzeczności (decyzje merytoryczne)

| ID | Kwestia | Rekomendacja |
|----|---------|--------------|
| K1 | „Rozrywanie” (Tech. 10) vs „redukcja / uwodornianie mostków” (Mat. 38–39) | Uczyć **obu warstw**: język egzaminacyjny Tech. + precyzja Mat. w feedbacku. Nie mówić, że włos „znika” lub jest „zniszczony całkowicie”. |
| K2 | Mostki wodorowe vs wiązania wodorowe | Synonimy dydaktyczne; w tekście lekcji: **wiązania wodorowe (mostki)**. |
| K3 | Mostki siarczkowe vs wiązania siarkowe / disulfidowe | W źródłach: siarkowe / siarczkowe. **Nie wprowadzać** „disulfidowe” jako obowiązkowego, o ile nie ma w lokalnych źródłach (masterplan używał słowa roboczo — tu ostrożnie). |
| K4 | Mat. 19 (ondulacja wodna) vs Tech. 6/18 (wiązania H) | Spójne: trwałość ułożenia zależy od procesu modelowania/wysuszenia; mechanizm = wiązania H. |
| K5 | PDF „materiały do egzaminu…” bez OCR | Nie cytować procentów/definicji wyłącznie z tego pliku, dopóki nie ma visual review. |
| K6 | Wzmianka o utrwalaczu w module pH | W module wiązań **pogłębiamy** (kolejność: reduktor → kształt → utrwalacz/utlenianie), nie udajemy, że uczennica nigdy nie słyszała słowa „utrwalacz”. |

## 6. Mapa modułu (ustalona liczba lekcji)

**7 lekcji** — wynika ze źródeł (Tech. 6, 10, 18 + Mat. 6, 19, 22, 38–40), bez tematów niepokrytych.

| # | id (propozycja) | Tytuł | Jedna główna myśl | Główne źródła |
|---|-----------------|-------|-------------------|---------------|
| 1 | `wlos-ksztalt-co-utrzymuje` | Co utrzymuje kształt włosa? | Kształt włosa utrzymują wiązania w strukturze keratyny — nie „pamięć lakieru” | Keratyna (wcześniej) + wprowadzenie do Tech. 6/10/18 |
| 2 | `wlos-ksztalt-wodorowe` | Wiązania wodorowe | Wiązania wodorowe (mostki) łączą elementy struktury i reagują na warunki modelowania | Tech. 6; plansza H |
| 3 | `wlos-ksztalt-wilgoc-cieplo` | Wilgoć, ciepło i modelowanie | Przy modelowaniu wilgoć i ciepło powodują **czasową** zmianę wiązań wodorowych | Tech. 6; Mat. 19 |
