## Zakres do przyszłego modułu lekcji

- Czym jest K-PAK (rekonstrukcja fryzjerska vs „leczenie”)
- Model O-W-R-N (oczyszczanie → wygładzanie → rekonstrukcja → nawilżenie)
- Procedura produktowa z instrukcji (czasy aplikacji, ręcznik, unikaj skóry głowy)
- Powtórzenie kroków 2–3 przy mocno uszkodzonych włosach; tonacja/koloryzacja po rekonstrukcji
- Keratyna / aminokwasy — język dla klientki
- SmartRelease, Quadramine — bez wykładu chemicznego
- Gotowe wypowiedzi 15/30 s; czego nie obiecywać
- Kiedy proponować (po rozjaśnianiu, szorstkość, przed koloryzacją)

## Uwagi

- Włos na długości = martwa struktura — zabieg działa kosmetycznie/fryzjersko.
- Nie porównywać marketingowo z Olaplexem („tylko lepsze”).
- Preferować `…ladniejsza-wersja.pdf` przy budowie scenariuszy lekcji; procedurę trzymać zgodnie z `…instrukcja-wykonania.pdf`.
